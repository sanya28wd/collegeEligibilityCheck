This project is a Java Swing–based desktop application that helps students find suitable engineering colleges based on their academic performance, entrance exam scores, budget, ratings, and preferred state.

The application provides an interactive graphical user interface (GUI) where users first log in and then enter their academic details such as board marks, entrance exam marks, preferred state, maximum budget, and desired college rating. Based on the provided inputs, the system filters and sorts a dataset of engineering colleges and displays the results in a structured table format.

The project demonstrates the use of Java Swing components, event handling, object-oriented programming, and data filtering/sorting using Java Streams. It simulates a simple college recommendation system using predefined sample data.

FEATURES
Login Interface with validation (username, age, password)
User Input Form to enter academic scores and preferences

FILTERING OPTIONS BASED ON:
1. Board Marks
2. Entrance Exam Marks
3. State Preference
4. Maximum Fees
5. College Ratings

SORTING OPTIONS FOR:
1. Ratings
2. Fees
3. Entrance Scores
4. State

Tabular Display of colleges using JTable and Java Streams & Comparators used for filtering and sorting. User-friendly GUI built using Java Swing.

TECHNOLOGIES USED:
1. Java
2. Java Swing (GUI)
3. Object-Oriented Programming (OOP)
4. Java Collections Framework
5. Java Streams API

LEARNING OUTCOMES
1. Building multi-window GUI applications in Java
2. Implementing event-driven programming
3. Using streams and comparators for data filtering and sorting
4. Managing data using custom classes and collections
